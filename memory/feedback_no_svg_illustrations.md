name: feedback-no-svg-illustrations
description: User prefers real images over hand-drawn SVG illustrations for visual hero/marketing sections
metadata:
  type: feedback
---

Do not attempt to draw custom SVG illustrations as hero images or decorative visuals. Use real provided images instead.

**Why:** SVG-drawn illustrations (stars, landmasses, etc.) look amateur compared to real artwork. When the user provides an image asset, use it.

**How to apply:** If a hero or marketing section needs a visual and the user provides an image, incorporate the real image. Ask the user to place image files in the project directory if they aren't already there — don't try to recreate the effect with SVG paths.
